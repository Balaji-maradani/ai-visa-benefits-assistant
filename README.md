# AI Visa Benefits Assistant (Prototype)

## Description
This project is a prototype of an AI-powered Visa card benefits assistant.
It helps users understand card benefits, offers, and terms clearly using
context-aware recommendations.

## Features
- Masked Visa card input
- Auto-formatted card number
- Benefit and offer display
- Best benefit recommendation
- Terms & conditions (demo data)
- Awareness-only prototype

## Tech Stack
- HTML
- CSS
- JavaScript

## Note
This is a demonstration prototype using mock data.
No real Visa card data is stored or processed.
